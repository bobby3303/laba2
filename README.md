# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего цифры от 1 до 9, получить список из их двоичных значений.

### Алгоритм решения

Вводим список, проверяем данные на корректность. При помощи List.map каждое число исходного списка из десятичного перобразуем в двоичное классическим алгоритмом деления столбиком и записи элементов задом наперед. Реализуем алгоритм перевода числа в двоичную СС с помощью рекурсии и деления, с последующим приписыванием остатков в виде строки.

# Задание 2

## Задача 1

### Текст задачи

Список содержит десятичные цифры. Получить строку, перечисляющую эти цифры. Например, для списка [1; 3; 1; 7] получаем строку «один; три; один; семь»

### Алгоритм решения

Вводим список цифр, проверяем на корректность. С помощью List.fold каждую цифру исходного списка переводим в слово и на основе этого создаем строку, которая во время обработки элементов списка будет увеличиваться
